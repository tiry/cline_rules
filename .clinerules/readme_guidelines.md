
## README Guidelines

The root folder must include a `README.md` with:

* A description of the project
* The overall project structure
* Installation and usage instructions
* Build and test commands

Keep this file **up to date** throughout development:

* Update the project structure section as files are added/removed.
* Reflect changes to installation or test steps promptly.

If `README.md` exceeds **200 lines**, split it into the following files:

* `Usage.md` — for users of the project
* `Install.md` — build and install instructions
* `Dev.md` — development setup and testing instructions
